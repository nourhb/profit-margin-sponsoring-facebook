# **App Name**: ProfitMargin

## Core Features:

- Input Product Economics: Collect data like price, COGS, shipping, packaging, COD fee, and other expenses.
- Ad Spend and Conversions: Collect the number of purchases attributed to an ad, cancellations, and ad spend.
- Margin Before Ads Calculation: Calculates margin before ads and margin percentage to visualize the gross profit margin before advertisement costs are factored in.
- Net Purchases and CPA Calculation: Calculates the number of net purchases, and CPA (cost per action) for running ads.
- Profitability Decision: Use all data inputs to calculate and communicate in a friendly way whether a profit can be realized or not, for the inputs entered by the user.
- Profit Target Analyzer: Specify a target profit to project and help make decisions based on ads spend.

## Style Guidelines:

- Primary color: Dark grayish-blue (#5A698F) for a professional yet approachable feel.
- Background color: Light gray (#F0F2F5) for a clean, neutral backdrop that doesn't distract.
- Accent color: Soft lavender (#A096D3) for subtle highlights and interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) to create a modern, yet still warm and accessible, style.
- Use a set of clean, minimalist icons for key metrics and input fields to aid visual understanding.
- Ensure consistent spacing and padding across all elements to enhance readability and user experience.
- Subtle animations can be used to highlight calculation results and provide feedback on user interactions.
